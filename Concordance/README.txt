Files Submitted :
1. sample.txt-The file to be read by the program
2. Java files :
		Word.java- Represents a class that will maintain the count and line number list for each word read.
		Main.java - To read the sample.txt file and process each word read to print its countand occurences line number


How to compile:

1. Open Project using Intellij/Eclipse.
2. Build the project and run the Main.java class.


Description :

In the program, we maintain the count of each word read using an object for each word read and putting the word and the object
into a TreeMap to maintain the alphabetical order.
